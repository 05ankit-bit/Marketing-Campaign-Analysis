# Marketing-Campaign-Analysis

# Project Overview
This project focuses on identifying the most effective marketing channel and reassessing the marketing spend based on KPIs like ROAS(Return on ad spend), ROI(Return on Investment) and CAC(Customer Acquisition Cost).

## Objectives
- To find out the most profitale marketing channel
- To find out which channel needs to be scaled and which needs to be discontinued
- To find out the stage having maximum drop off in funnel conversion
- To find out important KPIs like CAC, ROAS and ROI for each channel
- To find out whether marketing money is spend effectively or not

## Tools & Technologies Used
- MS Excel for understanding the data
- MYsql workbench for data cleaning and data validation
- Power Bi for data modelling, visualization and dashboard development

## Dataset Information
- The dataset contains 11 tables. 7 dimension tables and 4 fact tables containing columns like;
- Marketing spend
- Gross revenue
- Net Revenue
- Discount
- Impressions
- Clicks
- Sessions
- Product views
- Purchases
- Customer ID
- Campaign ID

## Key Insights
- Meta ads and Affiliated marketing have highest ROAS but also high CAC
- Maximum drop off is between product views and add to cart stage of Conversion funnel
- Conversion % of Purchasing funnel is 6.9%
- Conversion % of Advertising funnel is 0.3%
- New customers contributes more in revenue while returning customers contribute less
  
## Dashboard Features
- Revenue trend Line chart
- Bar Chart for Channel vs ROAS
- Bar Chart for Channel vs Marketing spend
- Funnel chart for Purchasing funnel and Advertising funnel
- Pie Chart for revenue contribution of New vs Returning customers
- Interactive Slicers

## How to Run the Project
1. Clone this repository
2. Open .sql file in MYsql workbench for data cleaning and validation steps
3. Open .pbix file for visuals and business insights
